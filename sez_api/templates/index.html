Jak funguje hromadné ztotožnění?
1. Připravíte CSV soubor s údaji pacientů (jméno, příjmení, rodné číslo / datum narození).
2. Nahrajete soubor — systém vrátí ID žádosti.
3. Spustíte zpracování (vykonání) — systém asynchronně ztotožní každého pacienta proti registru.
4. Zkontrolujete stav a stáhnete výsledky (CSV/JSON) — u každého pacienta uvidíte přidělený RID a stav ztotožnění.
Volitelně lze při podání žádosti registrovat odběr změn dat u ztotožněných pacientů.

Sloupce: jmeno · prijmeni · rodneCislo · datumNarozeni (YYYY-MM-DD) · cisloDokladu · typDokladu (OP, PAS…) · datumUmrti

💻 Ukázky kódu – Base64 a Hash

Jak generovat Base64 z PDF a spočítat SHA-256 hash v různých jazycích pro volání EZCA API.

Python
C# / .NET
JavaScript
IRIS ObjectScript
cURL / Bash
import base64

# PDF soubor → Base64
with open("dokument.pdf", "rb") as f:
    obsah_b64 = base64.b64encode(f.read()).decode("utf-8")

# Použití v EZCA API requestu
request_body = {
    "obsahDokumentu": obsah_b64,
    "nazevSouboru": "dokument.pdf",
    "typPodpisu": "PADES"
}

# Base64 → zpět do PDF
pdf_bytes = base64.b64decode(obsah_b64)
with open("podepsany.pdf", "wb") as f:
    f.write(pdf_bytes)

# SHA-256 hash souboru (pro EZCA podpis/razítko hashe)
import hashlib
with open("dokument.pdf", "rb") as f:
    sha256_hex = hashlib.sha256(f.read()).hexdigest()
    print(sha256_hex)  # → vložit do EZCA "hashHodnota"

# Hash z Base64 obsahu
raw = base64.b64decode(obsah_b64)
sha256_hex = hashlib.sha256(raw).hexdigest()
// PDF soubor → Base64
byte[] pdfBytes = File.ReadAllBytes("dokument.pdf");
string obsahB64 = Convert.ToBase64String(pdfBytes);

; PDF soubor → Base64
ClassMethod FileToBase64(cesta As %String) As %String
{
    Set stream = ##class(%Stream.FileBinary).%New()
    Set sc = stream.LinkToFile(cesta)
    Set b64stream = ##class(%Stream.TmpBinary).%New()

    ; Kódování po blocích (pro velké soubory)
    While 'stream.AtEnd {
        Set chunk = stream.Read(57 * 100)  ; násobek 57 pro čisté řádky
        Do b64stream.Write($System.Encryption.Base64Encode(chunk))
    }

    Set result = ""
    Do b64stream.Rewind()
    While 'b64stream.AtEnd {
        Set result = result _ b64stream.Read(32000)
    }
    Quit result
}

; Jednodušší varianta (malé soubory do ~2MB)
ClassMethod PdfToBase64Simple(cesta As %String) As %String
{
    Set stream = ##class(%Stream.FileBinary).%New()
    Do stream.LinkToFile(cesta)
    Set data = stream.Read(stream.Size)
    Quit $System.Encryption.Base64Encode(data)
}

; Base64 → zpět do PDF
ClassMethod Base64ToFile(b64 As %String, cesta As %String) As %Status
{
    Set data = $System.Encryption.Base64Decode(b64)
    Set stream = ##class(%Stream.FileBinary).%New()
    Set sc = stream.LinkToFile(cesta)
    Do stream.Write(data)
    Quit stream.%Save()
}

; SHA-256 hash souboru
ClassMethod FileHash(cesta As %String) As %String
{
    Set stream = ##class(%Stream.FileBinary).%New()
    Do stream.LinkToFile(cesta)
    Set data = stream.Read(stream.Size)
    Set hash = $System.Encryption.SHA256Hash(data)
    ; Převod na hex řetězec
    Set hex = ""
    For i=1:1:$Length(hash) {
        Set hex = hex _ $Translate($Justify($ZHex($Ascii(hash,i)),2)," ","0")
    }
    Quit $ZConvert(hex,"L")
}

    }
